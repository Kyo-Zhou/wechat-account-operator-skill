# Quality Bar

Use this as the final review gate.

## Content

The draft should:

- answer a real reader question
- contain a clear point of view
- explain why the event matters
- avoid empty trend language

## Writing

The writing should:

- open fast
- avoid slogan-like lead paragraphs
- keep section logic obvious
- end with a usable judgment

## Layout

The layout should:

- look edited for WeChat
- make scanning easier
- use emphasis intentionally
- avoid looking like a tool preview

## Not Ready Yet If

The draft still feels:

- generic
- overdesigned
- under-edited
- too dense
- too AI-sounding

If any of those are true, revise before treating it as publishable.
